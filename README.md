# Battery State of Charge (SoC) Estimator

A Python-based project to estimate battery State of Charge using the Coulomb Counting method.

## Features
- Estimates remaining battery SoC
- Uses current and time-based discharge calculation
- Beginner-friendly BMS concept
- Input validation

## Tech Stack
- Python 3
